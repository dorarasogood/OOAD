# 餐廳管理系統(Restaurant Management System)

類似POS系統。<br>
這個系統角色主要有三種，顧客、員工、老闆。顧客可以使用這套系統點餐，透過系統的菜單介面，讓顧客可以快速瀏覽菜單並點餐，同時可以顯示座位清單，讓顧客可以選擇自己想要的座位;員工可以快速接收到顧客訂單並提供服務，同時也可以依照座位清單幫預約顧客保留座位，老闆可以使用這套系統檢視餐廳的營運情況。

<br>
